# Natural Bank - Banking Reimagined

Welcome to **Natural Bank**, a next-generation online banking platform designed with a focus on aesthetics, user experience, and modern technology.

## Features

-   **Stunning UI/UX:** Glassmorphism design, smooth GSAP animations, and interactive elements.
-   **Account Opening Flow:** Seamless multi-step process to open Savings or Current accounts with real-time validation.
-   **Dashboard:** View balance (PIN protected), quick actions, and financial insights.
-   **Secure:** Mock backend implementation with proper validation structure and secure patterns.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

-   **Framework:** Next.js 14+ (App Router)
-   **Styling:** Tailwind CSS
-   **Animations:** Framer Motion, GSAP
-   **Database:** MongoDB (Mongoose)
-   **Icons:** Remix Icon, Lucide React
